Ce projet contient un jeu de type "angry birds" fait avec le moteur Unity 5.4 puis mis à jour sur Unity 2017. 
Les sources du jeu sont contenues dans le dossier "source". Vous pouvez également les importer avec le unitypackage "MyAngryBirds".
Si vous voulez lancer le jeu sur PC, il faut télécharger le dossier "versionPC" et lancer le .exe présent à l'intérieur.
Si vous voulez lancer le jeu sur Android, il faut avoir un Smartphone ayant Android 4.4 au minimum. Il faut ensuite mettre le fichier apk dans son smartphone et l'installer.
